 Clasificador Iris con Machine Learning

Este proyecto implementa un modelo de clasificación utilizando el famoso dataset Iris.

 Tecnologías utilizadas
- Python
- scikit-learn
- K-Nearest Neighbors (KNN)

 ¿Qué hace el modelo?

Clasifica flores en tres especies:
- Setosa
- Versicolor
- Virginica

Basándose en:
- Largo del sépalo
- Ancho del sépalo
- Largo del pétalo
- Ancho del pétalo

 Cómo ejecutarlo

1. Instalar dependencias:
pip install scikit-learn

2. Ejecutar:
python iris_modelo.py

 Resultado

El modelo logra aproximadamente 100% de precisión en el conjunto de prueba.
